Quick Start Guide
  1. Choose a system from the dropdown at the top (e.g., Fermentation Vat).
  2. Set PID gains using the Kp, Ki, Kd sliders on the right. Start with Kp only (leave Ki and Kd at 0).
  3. Press ▶ Start to run the simulation, then Step SP to change the target value and observe the response.
  4. Watch the time chart (bottom-left) to see how the process variable tracks the setpoint. The effort chart (top-right) shows each PID term's contribution.
  5. Adjust gains while the simulation runs to see immediate effects. Try adding Ki to eliminate steady-state error, then Kd to reduce overshoot.

Tips: Use the speed selector (top bar) for slow systems like the vat. The Disturb button injects a random perturbation to test disturbance rejection. Toggle PID View to see generic control-theory labels. Try Manual mode to control the actuator by hand — then appreciate why PID exists!

Assignment Mode: Click Assignment Mode, enter your Canvas variables A and B, select your problem, and tune the PID gains until all performance goals are met. Your completion code will appear automatically.
